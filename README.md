# API Hamburgueria

Projeto desenvolvido para a disciplina de Back-End com o objetivo de implementar uma API REST utilizando Node.js, Express e Sequelize.

## Funcionalidades

- Cadastro e listagem de categorias
- Cadastro e listagem de produtos
- Cadastro e listagem de pedidos
- Cadastro de entregas
- Avaliação de pedidos (nota de 1 a 5)
- Relacionamentos entre as entidades
- Uso de eager loading nas consultas

## Tecnologias

- Node.js
- Express
- Sequelize
- MySQL
